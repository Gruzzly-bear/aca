SCP-9086

Item #: SCP-9086
Object Class: Neutralized
Site Responsible: Site-111
Director: Graham Young
Research Head: Alice Hawkins
Assigned Task Force: None
Level 3/9086
CLASSIFIED

Photograph of an Intel Paragon system similar to SCP-9086.

Special Containment Procedures: This document serves to archive information relating to the object designated SCP-9086. As of roughly 03:45:00 UTC on 2025-12-27, SCP-9086 ceased communication and was later considered neutralized. SCP-9086 does not require additional special containment procedures.

Description: SCP-9086 appears to be an Intel Paragon supercomputer that has fallen into disrepair, resembling systems produced by Intel in the 1990s. SCP-9086 was acquired during a foundation raid on an abandoned building located near Cambridge, Massachusetts, following the spread of rumors regarding the existence of a previously undocumented supercomputer on deep web forums. SCP-9086 contained a program designated "Mira". Analysis suggests the "Mira" program could have been used to host and execute an advanced artificial intelligence. A large log file (See Addendum) was also discovered on SCP-9086. However, both the log file and "Mira" program appear to have experienced various levels of data loss and corruption due to failure in the device's memory. SCP-9086 is considered neutralized as SCP-9086 is currently inoperable. Additionally, the "Mira" program is corrupted beyond recovery and can no longer be executed.
