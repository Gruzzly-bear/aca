Special Containment Procedures: SCP-9985 is to be stored in a locker at Storage Site-64. All personnel wishing to use SCP-9985 for experimental use are to request permission from Dr. Pien. Under no circumstances can SCP-9985 be used for personal any use. A copy of SCP-9985-2 is to be stored on a Foundation server.

Description: SCP-9985 is a ██████ brand desktop computer, with a preloaded operating system named "StallOS" (designated SCP-9985-1), a Debian-based Operating system. The OS functions as a standard Linux machine; opening the "Staller.sh" file (designated SCP-9985-2) will result in an input menu appearing with the text "Please enter your delay:". Upon entering an upcoming event, the specified event will inevitably be delayed by various means; attempts to run events on schedule result in personnel being late, airplanes crashing, [DATA EXPUNGED]. SCP-9985-2 appears to only work on StallOS, and delays vary in time. Attempts to decompile SCP-9985-2 are ongoing; progress is limited by complex encryption..

Addendum [9985a] (Retrieval): SCP-9985 was recovered by Agent ███████ during a containment breach involving SCP-████. SCP-████ was using SCP-9985 to delay its recapture. SCP-████ was detained before successfully entering a prompt into SCP-9985-2. Out of curiosity, Agent ███████ entered ‘█████ █████’s Concert,’ after which the concert was delayed due to weather. Origins remain unknown.

Addendum [9985b]: The query "Taking my pills" was typed into SCP-9985-2 by Dr. Pien, after which Dr. Pien proceeded to not take medication for his cardiac condition. Dr. Pien reported that he felt no adverse effects.
Notes: Useful. - Dr. Pien

Addendum [9985g]: The text "D-77478 dropping a ball" was typed into SCP-9985-2, after which Dr. Pien instructed D-77478 to drop the rubber ball on the table. D-77478 suddenly became irate, refused to follow orders, and attacked the security guard on standby, resulting in D-77478's death.
Notes: Requesting access to SCP-9985 for testing. - Dr. Raker

Addendum [9985h]: Dr. Raker, in an attempt to force an event to occur as scheduled, typed in "Dr. Raker will head to room 204". Dr. Raker proceeded to walk through the hallway; just before opening the door to room 204, Dr. Raker suddenly collapsed due to a stroke. Dr. Raker made a full recovery after four weeks.
Notes: This incident tells us that if an event is hard to be delayed, SCP-9985-2 will just disrupt them. Further testing to forcefully run events is prohibited.
Notes (2): Seriously? Only one time and it's banned? Requesting repeat testing. - Dr. Raker
Notes (3): Denied. Next time it'll be a fucking heart attack, Tim. - Dr. Pien

Addendum [9985k]: On 4/22/████, a new application named "WORM5C.sh" (designated SCP-9985-3) was added to SCP-9985-1via unknown means, and cannon edited or deleted. Running the file shows text "WARNING - DONT LOOK AT IMAGES :)". Personnel are advised not to run the application.
Notes: Requesting permission to test SCP-9985-3. - Dr. Wagner
Notes (2): Granted. Be careful. - Dr. Pien
Notes (3): Yeah yeah… - Dr. Wagner
